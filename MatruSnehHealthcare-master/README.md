# MatruSnehHealthcare

MatruSnehHealthcare is an Android application developed to support maternal and child healthcare services through a simple and user-friendly digital platform.

## Features

- User-friendly Android interface
- Maternal healthcare support
- Health-related information and services
- Organized and responsive application design
- Easy navigation for users

## Technologies Used

- Android Studio
- Kotlin
- XML
- Gradle

## Project Structure

- `app/` – Main application source code
- `res/` – UI resources and layouts
- `manifests/` – Android manifest files
- `Gradle Scripts/` – Build configuration files
