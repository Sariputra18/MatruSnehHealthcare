\# MatruSnehHealthcare



\## Overview

MatruSnehHealthcare is an Android healthcare application designed to support maternal health monitoring and wellness tracking. The app helps mothers manage health records, nutrition logs, kick tracking, and profile information through an easy-to-use digital platform.



\## Features

\- Mother profile management

\- Nutrition tracking

\- Baby kick monitoring

\- Local data storage using Room Database

\- MVVM architecture implementation

\- Simple and user-friendly UI



\## Tech Stack

\- Android Studio

\- Kotlin

\- Jetpack Compose

\- Room Database

\- MVVM Architecture

\- Material Design



\## Purpose

This application aims to improve maternal healthcare awareness and provide a digital companion for health monitoring.
